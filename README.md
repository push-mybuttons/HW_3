# HW_3

Проект для автоматизированного тестирования.

## Технологии
- Java 17
- JUnit 5
- Selenide
- Gradle

## Структура проекта
- `src/main/java` - исходный код
- `src/test/java` - тесты
- `src/main/resources` - ресурсы
- `src/test/resources` - тестовые ресурсы

## Запуск тестов
```bash
./gradlew test
```
